## [WORLD]
Name: Caelos. 12 rectangular regions (political borders). Active regions: Calmaar (central, current campaign), Caldorn (NW), Calmeri (N), Calvyce (S). 8 undeveloped regions pending. Era: Industrialis Developmentia (I.D.). Year: 1219 I.D. Early-mid industrial revolution meets high fantasy magic. Magic exists but less culturally central than previous era — industry filled the vacuum. Tone: morally grey, wonder, mystery, dark moments balanced with humor. Consequences permanent and world-altering. Two active parties playing simultaneously in same world — actions affect each other. Both player groups know the other party exists out-of-game. In-game awareness minimal.
Party A = Heroes of Hayward (named after saving the Artisan Festival from Steam Mephits). Party B = The Starfall Society (meteors/starfall origin). Colloquially still referred to as PA/PB for shorthand. Original music by DM (Vorpal Lance Music). In-world newspaper (Word of the Whelk) documents world events in real time. Long-term vision: both parties meet in large multi-player session once timelines align. Future: undeveloped continent across Abyssal Shelf ocean to the west — golden city on its shores. Lifetime legacy project.

## [DASHBOARD] Caelos DM Dashboard — Technical Reference
File: caelos_dm_dashboard.html | Single-file HTML PWA | No build tools | ~2.3MB
Hosting: GitHub Pages. Target: Pixel 10 Pro (mobile-first).
Fonts: Rajdhani (headers) + Nunito (body). Key vars: --gold #e8c87a, --blood #9b1a1a, --surface #12121a, --raised #1a1a26.
Architecture: 2 script blocks. Script 1: portrait base64 data. Script 2: BESTIARY array + all JS logic.

### TABS (5)
1. COMBAT — round tracker, Surprise Round toggle, party HP blocks, Conditions, Enemies.
2. BESTIARY — card grid + creature detail view.
3. ENCOUNTER tab — pending repurpose to CODEX (Reference/NPCs/World Notes/Dice Roller). Not yet built.
4. PREP — fully built. localStorage persisted.
5. WEATHER — existing.

### PARTY SYSTEM
Badge top-right toggles Heroes of Hayward (A) / The Starfall Society (B).
partyName(p) helper returns full name. toggleParty() fires renderPCs() + refreshes Prep if active.
Prep localStorage keys namespaced: caelos_prep_A_ / caelos_prep_B_.

### BESTIARY TAB
8 creatures fully statted + portrait embedded. BESTIARY array + PORTRAITS map in Script 1.
Card grid: 2-col, face-down default. Tap: 3D flip + gold spark burst. CR10+ get holo-foil shimmer.
Flip toggle: face-up card tapped again flips back. Flipped cards glow gold.
Card back: creature name + CR (not "Caelos Bestiary"). Sheen 9s staggered. Sigil breathe 6s.
Detail view: portrait hero > AC/HP/Speed/Init > ability grid > 4 tabs (Stats/Actions/DM/Lore).
Deploy button: adds to activeEnemies[], switches to Combat, opens Enemies section.

### ENEMY TRACKER (Combat > Enemies section)
Per-group: portrait thumb + name/type/CR + stats strip (AC, Max HP, Speed, Pass.Perc, STR/DEX/CON saves).
HP bars: green>orange>red with scan shimmer. Per-instance damage/heal inputs.
Header glows red when enemies active. Reminder chips: passive (blue) / triggered (pulsing red).
Gertie special: "💚 Keep my girl safe at ALL COSTS! 💚"
Functions: deployToCombat(id), dismissEnemy(gi,ii), dismissGroup(gi).

### PREP TAB (localStorage — all persisted across sessions)
Global Notes: party-agnostic. Key: caelos_prep_global. Blue-tinted. Always at top.
Party banner: pulsing red, shows current party, animates on switch.
Per-party sections (swap on party toggle):
  - Pinned Reminders: caelos_prep_{A|B}_pins. Gold-tinted. Enter or + to add.
  - To-Do: caelos_prep_{A|B}_todos. Tap to check. Badge = remaining. Clear Completed / All.
  - Session Notes: caelos_prep_{A|B}_notes. Auto-saves with timestamp. Archive > pushes to Recap + clears.
  - Recap Log: caelos_prep_{A|B}_recaps. Dated, newest first, collapsible, deletable. Badge = count.
